## 🚀 如何运行 E2E 测试

### 快速开始
```bash
# 1. 启动后端（终端 1）
cd packages/server
bun run dev

# 2. 启动前端（终端 2）
cd packages/admin
bun run dev

# 3. 运行 E2E 测试（终端 3）
cd packages/admin
bun run test:e2e
```

### 查看测试报告
```bash
bunx playwright show-report
```

**详细指南**: 参见 `packages/admin/e2e/E2E_TEST_GUIDE.md`

### 1. 租户路由 Mock 问题（23 个测试失败）
**原因**: Mock DB 不支持路由参数（`:id`）查询

**影响**: 低 - 功能已手动验证，中间件测试 100% 通过

**修复计划**: 
- 增强 Mock DB 的 `first()` 方法
- 或使用集成测试框架（Supertest）
- 优先级：低
